New Features of javascript

There are a lot of new features for programmers in javascript ES6 as compare to ES5. I am going to discuss some of the important features of ES6.

**1.Expression Closures: ** Expression Closure is the combination of function and lexical environment in which the function has been defined. It provides simple and shorthand syntax to write functions in Java script.

Let’s see the Example of Expression Closures: function first(x) { return function second () { return x+1; } }

var func = first (5); func ();

2.Default parameters: Java script provides feature of default parameter that is a better way to make your methods well organized and shorter. I found that use of default parameters is being more productive and conserve less time.

**Example:**
    function multiply (a, b = 2) {
     return a * b;
    }
    multiply (5); // 10

    function dblmultiply (num = 1, multi = multiply(num)) {
     return [num, multi];
        }
    dblmultiply (); // [1, 2]
    dblmultiply (6); // [6, 12]

3.Template Literals: These are string literals which allow the embedded expression in your code. User can use multi line strings with template literals.

Syntax: string text string text line 1 string text line 2 string text ${expression} string text

Template literals are enclosed by the back-tick (` `) instead of single or   double quotes. Template literals are defined by dollar sign and curly braces(${expression}). Theses expressions will be passed to a function and concatenates the parts into a single string.

4.Multiline strings: New java script has easy way to declare multiple line strings by using backticks that removes the confliction for developers and save their time. This is a part of template literal. Let’s see the syntax and example to get multi-line strings using normal strings.

Syntax: console.log (‘string text line 1\n’ + ‘string text line 2’); Example: Console.log (I am student of Langara\n doing mobile app design and web development program)

To get the same string into multi line strings, you can use the following: syntax and example: console.log (string text line 1 string text line 2);

**Example:
Console.log (`I am student of Langara doing mobile app design and web development program `)**

5.Destructing Assignment: Java Script supports destructing assignment that provides a best and convenient way to extract values from data stored in nested objects and arrays. Destruction allow you to use patterns to extract the data such as the left-hand side of an assignment.

Let’s see the Example to use destructing in variable declaration. Let obj = {first: ‘Kulwinder’, last: ‘Kaur’}; Let {first: f, last: l} = obj;

Here, I declare the variable f and l and assigns them the values ‘Kulwinder’ and ‘Kaur’. Destructing can be used in statements below:

The variable initialization can be done by using any type like Let, Const and var. Let [x] = [‘k’]; Const [x] = [‘k’]; Var [x] = [‘k’];

[x] = [‘k’]; Function f([x]) { … } F([‘k’]);

To completely understand what destructing is, let’s see the example by comparing constructing and destructing.

Firstly, see the constructing syntax:

Let ob = {}; ob. First = ‘Kulwinder’; ob. Last = ‘Kaur’;

Another way we declare the constructing is called an object literal like: Let ob = {first: ‘Kulwinder’, last: ‘Kaur’};

Now using extracting operation on data, we will define it as: Let f = ob. First; Let l = ob. Last;

And destructing enables same syntax for extracting as: Let {first: f, last: l} = ob

6.Enhanced object literals: Object literal is one of the best and most powerful features in java script that replaces the XML. This a superset of JSON, which is now provides standard for data transport for data transport on the web. The best feature of object literals is it makes easy to assemble the nested set of data into a human- readable form.

Let’s consider the object literal into ES6 java script code:

Function getpc (RAM, Processor, Price) { Return { RAM Processor, Value }; }

In ES5 this is declared as: Function getpc (RAM, Processor, Price) { Return { RAM: RAM, Processor: Processor, Value: Value }; }

By comparing these two example you can see that ES6 syntax is simple and easy to understand by being less repetitive with removal of colon (:) and use of common variable.

7.Arrow Functions: Arrow function is best feature of java script. It makes your code concise and simplify the function scoping. It is a one-line mini function which use a new token, =>. Arrow functions are used in lambda’s java script and also in other languages. Let’s see the example of arrow functions in ES6 and compare it with ES5:

Code in ES5: Var multiply = function (x, y) { return x * y; };

Code in ES6: Var multiply = (x, y) => {return x * y};

By using arrow function, we can avoid the use of the function keyword.
